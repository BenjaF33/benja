¿Que es FlexBox?

Flexbox es un sistema de diseño en CSS que te permite organizar elementos en una página web de manera flexible y eficiente. Facilita alinear, distribuir y ajustar el espacio entre los elementos, tanto horizontal como verticalmente, sin tener que usar flotadores o posiciones fijas. Es ideal para crear diseños responsivos.
